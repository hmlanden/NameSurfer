Name Database Visualizer

View project onGitHub

This was an assignment that I did for class. I decided to not include the data file necessary to make this program run, because it included data provided to me by my school, and I felt that it might be inappropriate to share. I promise it works! ^_-

This program reads in a datafile and creates a database that tracks the popularity of names over time. Users can input names and a visual representation of the trend will appear on the screen.

Note: I made use of the acm.libraries for graphics, since it was for school, and, as a friend of mine always tells me, trying to make a rectangle in Java without the acm is a nightmare, so they let us use it.
